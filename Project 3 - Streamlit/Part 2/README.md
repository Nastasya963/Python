# Анализатор данных 📊💰

Профессиональный инструмент для интерактивного анализа и визуализации данных из CSV-файлов, построенный на базе Streamlit.

### Описание
Программа предназначена для быстрого исследовательского анализа данных (EDA). Она позволяет пользователю загрузить структурированные данные и мгновенно получить визуальное представление зависимостей между признаками без написания кода.

**Основные возможности:**
- **Автоматическое определение типов:** Система пытается самостоятельно распознать типы данных, включая преобразование строковых представлений дат в объекты `datetime`.
- **Интерактивная визуализация:** Поддержка нескольких типов графиков:
    - *Линейные графики* — для отображения трендов и временных рядов.
    - *Диаграммы рассеяния (Scatter plot)* — для выявления корреляций между двумя числовыми переменными.
    - *Столбчатые диаграммы (Bar chart)* — для сравнения категориальных данных с использованием агрегации (по умолчанию — среднее значение).
    - *Гистограммы* — для оценки распределения плотности вероятности признака.
  
**Используемые алгоритмы и библиотеки:**
В основе визуализации лежат библиотеки `Matplotlib` и `Seaborn`. Для обработки данных используется `Pandas`, что позволяет эффективно работать с большими таблицами.

### Установка зависимостей
Для запуска программы на вашем компьютере должен быть установлен Python (версии 3.8 или выше).

1. **Клонируйте репозиторий или скачайте файл с кодом.**
2. **Создайте виртуальное окружение (рекомендуется):**
   ```bash
   python -m venv venv
   source venv/bin/activate  # Для Windows: venv\\Scripts\\activate
   ```
3. **Установите необходимые пакеты:**\
Создайте файл requirements.txt или выполните команду:
    ```bash
    pip install streamlit pandas numpy matplotlib seaborn
    ```
### Запуск
После установки всех зависимостей, запустите приложение из терминала, находясь в директории с проектом:
```bash 
streamlit run app.py
```
Для запуска на OS Windows перед названием файла необходимо поставить *точку и обратный слэш* (".\app.py")

После выполнения команды в вашем браузере откроется вкладка с интерфейсом приложения (обычно по адресу http://localhost:8501).
