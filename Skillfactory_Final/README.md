# 🌦️ WeatherInsight: Аналитическая панель погодных трендов

**WeatherInsight** — это интерактивное веб-приложение на базе Streamlit, предназначенное для визуализации и анализа исторических погодных данных из базы данных SQLite.

### 🚀 Функциональные возможности

1. **Фильтрация и управление данными:**
    + Календарный фильтр: Выбор произвольного диапазона дат для анализа.
    + Мультивыбор городов: Возможность сравнивать несколько городов сразу или просматривать данные по конкретному населенному пункту.
    + Автоматическая обработка типов: Преобразование строковых дат из SQLite в объекты Python datetime для корректной работы фильтров.

2. **Визуальный анализ:**
    + Общая статистика: Мгновенный расчет количества записей, уникальных городов и средней температуры за выбранный период.
    + Графики распределения:
    + Гистограммы: Визуализация частоты значений (температура, осадки, ветер) с наложением кривой плотности (KDE).
    + Boxplots (Ящики с усами): Сравнение разброса показателей по городам и поиск аномальных погодных явлений.

3. **Работа с таблицами:**
    + Интерактивный просмотр: Отображение данных через st.dataframe с возможностью сортировки по любому столбцу.
    + Постраничная навигация (Pagination): Быстрая работа с большими объемами данных благодаря разделению таблицы на страницы (настраивается слайдером).

### 🛠️ Технологический стек
- Python 3.x
- Streamlit: Каркас веб-приложения и интерфейс.
- Polars: Сверхбыстрая обработка и фильтрация данных (вместо Pandas).
- Plotly Express: Интерактивные графики.
- SQLite3: Хранение погодных данных.

### 📖 Как пользоваться приложением
1. Откройте боковую панель (Sidebar) слева.
2. Укажите интересующий вас диапазон дат в календаре.
3. Выберите один или несколько городов в списке.
4. В разделе "Анализ распределения" выберите показатель (например, Осадки) и тип графика, чтобы увидеть закономерности.
5. Используйте слайдер внизу боковой панели, чтобы настроить количество строк в таблице данных для удобного чтения.
